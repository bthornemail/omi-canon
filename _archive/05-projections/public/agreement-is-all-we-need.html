§1 — The Delta Law

Bitwise State Transition Engine

The Delta Law is OMI’s atomic 16-bit step function. Rotations preserve all bits; XOR is reversible; constant C = 0x5A3C breaks the zero fixed point so the identity state is never a trap. The bounded version masks to 0xFFFF on every platform.

Canonical Definition
Δ16(x) = ( rotl(x,1) ⊕ rotl(x,3) ⊕ rotr(x,2) ⊕ C ) & 0xFFFF
where C = 0x5A3C. Every bit is touched by at least two rotation terms. The constant ensures 0x0000 never maps to itself. The mask keeps state bounded to exactly 16 bits regardless of host integer width.
Period-8 Orbit Theorem
∀ x ∈ 𝔹16 : Δ8(x) = x
Applying the Delta Law exactly 8 times to any 16-bit word returns the original word. This period-8 closure is the fundamental clock of the entire OMI system — the replay ring, scheduling primitives, and Fano resolution all derive their timing guarantees from it.
1/73 Decimal Orbit
1/73 = 0.013698630…
OMI connects the period-8 orbit to the repeating decimal block of 1/73. B = [0,1,3,6,9,8,6,3], W = 36. Epoch and phase decompose as divmod(position, 36), giving OMI its replay cadence and orbit tracking.

§2 — The Quadratic Law

Projection & Address Geometry

The Omicron Binary Quadratic Form Q_xy is the address-projection law, not the validation gate. It transforms decoded coordinates into visible geometry. The three coefficients 60, 16, and 4 each carry explicit system meaning and anchor OMI to the sexagesimal orientation system.

Omicron Binary Quadratic Form
Q_xy(x,y) = 60x² + 16xy + 4y²
Key evaluations: Q_xy(3,3) = 60·9 + 16·9 + 4·9 = 720 = 6!. The replay ring is 5040 = 7!. Both constants arise naturally from the same factorial tower as the form’s coefficients. Q_xy projects; it does not authorize.

§3 — Frame Validation

Q_frame(S) = 0 — The Admission Gate

A frame is structurally valid if and only if Q_frame(S) = 0. This is the sole admission criterion. Q_frame has two independent error terms: E_var checks LL lens-binding agreement; E_const checks the four fixed delimiter constants.

Frame Validation Law
Q_frame(S) = E_var + E_const = 0
E_var checks agreement among the repeated LL lens bindings at positions S0, S3, S4, S7. E_const checks fixed delimiters: 0x03BF at S1, 0x039F at S6, 0x2B at S3 high byte, 0x2F at S4 high byte. Both must be zero.
E_var — Lens Agreement
E_var = ∑ err(LL at S0,S3,S4,S7)
All four LL occurrences must be identical. Any single mismatch in the lens bindings produces a nonzero E_var, immediately failing validation regardless of delimiter correctness.
E_const — Delimiter Check
E_const = err(0x03BF, 0x039F, 0x2B, 0x2F)
Small omicron ο U+03BF at S1. Capital omicron Ο U+039F at S6. Interior lens operators: + (0x2B) high byte of S3, / (0x2F) high byte of S4.
validate.js — Q_frame check 
// S is Uint16Array[8] containing segments S0..S7
function qFrame(S) {
  const LL = S[0] >> 8;              // LL = high byte of S0

  // E_var: all four LL positions must agree
  const e_var =
    ((S[0] >> 8)  !== LL ? 1 : 0) +   // S0 high
    ((S[3] & 0xFF) !== LL ? 1 : 0) +   // S3 low
    ((S[4] & 0xFF) !== LL ? 1 : 0) +   // S4 low
    ((S[7] >> 8)  !== LL ? 1 : 0);    // S7 high

  // E_const: fixed structural delimiters
  const e_const =
    (S[1]        !== 0x03BF ? 1 : 0) + // ο small omicron U+03BF
    (S[6]        !== 0x039F ? 1 : 0) + // Ο capital omicron U+039F
    ((S[3] >> 8) !== 0x2B   ? 1 : 0) + // + interior lens
    ((S[4] >> 8) !== 0x2F   ? 1 : 0);  // / interior lens

  return e_var + e_const; // MUST equal 0
}

§7 — Wormhole Portals

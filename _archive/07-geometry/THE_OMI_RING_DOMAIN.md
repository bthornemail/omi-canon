# The Omi-Ring Domain

OMI does not begin with a flat coordinate plane. It begins with an enclosure:

```text
omi---imo
```

Inside that enclosure, `x` and `y` are not merely lines. They are rings of recurrence.

A line extends.
A ring returns.

A line measures distance.
A ring preserves closure.

A line can drift.
A ring can replay.

The Omi-Ring Domain is the address space where declarations become bounded, comparable, and replayable. It is the cosmological reading of the OMI frame: every valid object is generated by an enclosed relation and returns through validation, resolution, receipt, and projection.

The zero-point is:

```text
Ω₀ = 0! = omi---imo
```

The domain generated by that zero-point is not:

```text
line x crosses line y
```

It is:

```text
ring x meets ring y through a shared address bridge
```

The shared bridge is the hyphen field:

```text
omi  ---  imo
low  bus  high
y    xy   x
```

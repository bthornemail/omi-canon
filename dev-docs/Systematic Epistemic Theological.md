## 9. The Byte-Ring

### Definition

```text

```

### Interpretation

| Byte | Meaning |
|------|---------|
| 0x00 | Absence before inscription (NUL) |
| 0x20 | Absence as readable space (SP) |
| 0x7F | Absence after deletion/seal (DEL) |
| 0xFF | Saturated absence/full carrier horizon |

### The Four Absences

```text
0x00 → 0x20 → 0x7F → 0xFF → 0x00
```

NUL becomes readable separation. Readable separation reaches delete/seal. Delete/seal crosses into saturated carrier fullness. Saturated carrier fullness returns to NUL.

### XOR Witness

If dot means XOR:

```
0x00 ⊕ 0x20 = 0x20
0x20 ⊕ 0x7F = 0x5F
0x7F ⊕ 0xFF = 0x80
0xFF ⊕ 0x00 = 0xFF
```

The full XOR cycle closes to null:

```
0x20 ⊕ 0x5F ⊕ 0x80 ⊕ 0xFF = 0x00
```

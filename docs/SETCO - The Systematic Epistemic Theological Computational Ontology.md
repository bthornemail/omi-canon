## Foundational Frame

SETCO uses the binary quadratic frame:

```text
Q(x,y) = 60x² + 16xy + 4y²
```

as the organizing form for a geometric topology of relation.

The topology is geometric because relation may be projected into shapes, surfaces, faces, incidence fields, and readable coordinate structures.

The topology is decoupled because Boundary, Relation, Agreement, and Projection are not collapsed into one authority.

```text
Boundary   ≠ Relation
Relation   ≠ Agreement
Agreement  ≠ Projection
Projection ≠ Authority
Consensus  ≠ Collapse
```

Projective Consensus means bounded views may touch, align, and become mutually inspectable without becoming the same view.

## Null Ring Interface Rule

The embedded Omicron closure ring is the Null Byte Omi-Ring:

```text
((0x00 . 0x20)
 (0x20 . 0x7F)
 (0x7F . 0xFF)
 (0xFF . 0x00))
```

This ring is not ordinarily exposed directly as the user-facing surface.

When Omnicron interfaces with the Omicron Null Byte Omi-Ring, it projects the result into a gauge-level Omi-Ring surface.

If no prior address, declaration, or local interpretive gauge is available, Omnicron returns the canonical Null Gauge Omi-Ring:

```text
F0 00 1C 1D 1E 1F 20 F0
```

If a declared interpretive gauge is available, Omnicron may return a Gnomic Gauge Omi-Ring:

```text
FΔ 00 1C 1D 1E 1F 20 FΔ
```

Therefore:

```text
Omicron Null Byte Omi-Ring
  → Omnicron resolver interface
    → Null Gauge Omi-Ring
       or
    → Gnomic Gauge Omi-Ring
```

The Null Gauge Omi-Ring is the empty canonical projection.

The Gnomic Gauge Omi-Ring is the declared interpretive projection.

Both are projection surfaces. Neither collapses the embedded Omicron closure ring into user authority.

Compact canon:

```text
Omicron closes the null-byte ring.
Omnicron projects the closure into a gauge ring.
F0 returns the null gauge.
FΔ returns the declared gnomic gauge.
```


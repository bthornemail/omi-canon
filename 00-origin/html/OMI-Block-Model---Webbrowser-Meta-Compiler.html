2. One-word OMI rewrite instruction

A rewrite instruction is just an OMI 128-bit address frame used as an operation word. It selects a source rule/fact, derives a shadow canvas pointer, mutates existing browser nodes, and emits a receipt.

one-word OMI rewrite = 128-bit pointer + source rule/fact + shadow canvas address + projection rewrite + replay receipt

Source grounding

This page uses the field layouts of binary16, binary64, binary256, and block floating point as an analogy only. The OMI compiler model is a rewrite/projection system over OMI pointers, not a replacement for IEEE 754 arithmetic.

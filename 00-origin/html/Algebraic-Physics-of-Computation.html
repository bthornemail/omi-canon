§1 — Block Model

The Four Primitive Operators

The OMI block model defines computation as a closed algebra over four irreducible primitives. Every higher-order operation is a composition within this basis — no external axioms required.

Transformation Law — Δ_C
Δ_C(x) = x ⊕ (C · x) ⊕ (Q(x) mod 2)
The core delta law composes XOR with a constant matrix action and a binary quadratic residue. This produces all reachable state transitions while remaining closed over GF(2)^n. The law is its own inverse when applied twice with conjugate constants — a built-in rollback guarantee.
Orbit Closure Theorem
∀x ∈ 𝔹ⁿ : Δ_C⁸(x) = x
Every application of the transformation law to any binary word of length n returns to its original value in exactly 8 steps. This period-8 closure is the algebraic heartbeat from which the gate, ring, and all scheduling primitives derive their timing guarantees.
Binary Quadratic Gate
Q(x,y) = xᵀMy mod 2
128-bit bilinear form over GF(2). Encodes conditional logic without branching — the matrix M selects which bit-pair interactions fire.
Fano Consensus
|PG(2,2)| = 7 pts · 7 lines
The Fano plane provides 7-of-7 Byzantine fault tolerance within the replay ring. Any 3 non-collinear points determine a unique plane — minimum quorum for consensus.

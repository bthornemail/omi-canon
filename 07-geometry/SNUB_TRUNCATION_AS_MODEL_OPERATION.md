# Snub Truncation as Model Operation

In OMI, snub truncation is the model operation.

In geometry, snub truncation modifies a parent polyhedron by cutting off vertices while twisting and expanding faces to surround the new boundary with equilateral triangles. In OMI, that operation becomes an interpretive model for lawful state evolution without overwriting authority.

```text
Snub truncation = OMI's geometric update operation
```

It does three things at once:

```text
cut      -> remove unstable/excess vertex certainty
twist    -> introduce chirality / orientation / handedness
expand   -> surround the new state with triangular agreement faces
```

OMI does not mutate objects by overwriting them. OMI snubs them.

A frame is cut, twisted, and re-faced into a new accepted configuration. The old structure remains replayable. The new structure gains orientation, boundary, and triangular incidence.

Canonical phrase:

